## 여는 글

당연하게도 사용자는 로딩이 빠르고 상호작용이 원활한 웹 경험을 원해요.

이를 위해 웹 프론트엔드 개발자가 브라우저에서 렌더링 되는 과정을 이해하고 어떻게 사용자가 더 좋은 경험을 할 수 있을지 고민하는 것은
어쩌면 당연하다고 볼 수 있어요.

웹의 성능에 있어 가장 주요한 문제는 다음과 같아요.

- 지연 시간
- 싱글 스레드

이 문제들이 무엇이고 어떻게 해결할 수 있는지 알아볼게요.

## 지연시간

**지연시간**은 빠른 로딩과 직결되는데 빠르게 요청하는 것과 빠르게 가져오는 것이 중요해요.

네트워크에서 발생하는 지연시간은 네트워크를 통해 컴퓨터로 바이트를 전송하는데 걸리는 시간이에요. 따라서 웹 최적화는 페이지 로드가 빠르게 이루어지도록 하는 방법들을 말한다고 볼 수 있어요.

## 싱글 스레드

브라우저의 대부분은 **싱글 스레드**이기 떄문에, 싱글 스레드의 동작방식을 이해하고 메인스레드의 책임을 줄여준다면 렌더링 시간을 향상 시킬 수 있어요.
