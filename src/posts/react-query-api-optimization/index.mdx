React Query를 사용하면 API 호출을 효율적으로 관리할 수 있습니다.
아래는 간단한 예시 코드입니다:

```tsx
const { data, isLoading, error } = useGetPostBySlug(slug);
```

`useGetPostBySlug` 함수는 포스트 데이터를 가져오는 함수입니다.

## 주요 기능

- 캐싱
- 자동 재시도
- 백그라운드 데이터 업데이트
- 등등...
